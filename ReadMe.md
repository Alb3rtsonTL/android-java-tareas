## Configuración del entorno

- **Android Studio** Meerkat 2025 o superior
- **JDK** 17 (embebido en Android Studio)
- **SDK mínimo:** API 24 (Android 7.0)
- **SDK objetivo:** API 36 (Android 16)

---

## Cómo abrir cualquier proyecto

1. Abrir Android Studio
2. `File → Open` → seleccionar la carpeta del proyecto
3. Esperar que Gradle sincronice
4. Conectar celular Android con **Depuración USB** activada
5. Presionar ▶ **Run**
